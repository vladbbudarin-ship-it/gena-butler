# Название задачи
Доработка мобильной версии сайта "Дворецкий Гена"

# Краткое описание задачи
Нужно было сделать мобильную версию удобной: убрать горизонтальную прокрутку, разделить список чатов и открытый диалог, исправить поле отправки, выбор срочности, мобильное меню, сообщения, вложения и читаемость кабинета Бударина.

# Что было изменено
- В пользовательском чате добавлено мобильное состояние: список чатов и открытый диалог.
- В открытом диалоге добавлена мобильная кнопка "← Чаты".
- На mobile список чатов теперь вертикальный, без горизонтальной прокрутки.
- Мобильное меню перенесено наверх, чтобы не перекрывать поле ввода и кнопку отправки.
- Поле отправки на mobile закреплено в нижней части чата и сохраняет textarea + кнопку отправки справа.
- Кнопки срочности на mobile размещаются в сетке и не требуют сдвига экрана.
- Добавлены mobile-правила для переноса длинных сообщений, имен файлов, вложений и кнопок.
- Добавлены ограничения overflow для основных mobile-экранов.
- Улучшена адаптация карточек и кнопок кабинета Бударина на узких экранах.

# Какие файлы были изменены
- `src/pages/MyQuestions.jsx`
- `src/index.css`
- `reports/codex/2026-05-16_23-09_mobile-layout-fix.md`

# Какие ошибки были найдены
- Список чатов на mobile был горизонтальной лентой.
- Список чатов и открытый диалог одновременно сжимались на одном экране.
- Мобильное меню располагалось около нижнего поля и могло перекрывать отправку.
- Кнопки срочности могли уходить в горизонтальный scroll.
- Вложения и длинные сообщения могли растягивать ширину экрана.

# Какие ошибки были исправлены
- Убран горизонтальный scroll списка чатов на mobile.
- Добавлен отдельный mobile layout списка и диалога.
- Меню вынесено в верхнюю часть экрана.
- Кнопка отправки выровнена и остается доступной.
- Срочность помещается в мобильную ширину.
- Добавлены переносы и ограничения ширины для сообщений, вложений и карточек.

# Какие проверки выполнены
- Проверены конфликтные маркеры через `git grep`.
- Выполнена production-сборка.

# Результат npm run build
Успешно. `vite build` завершился без ошибок.

# Что нужно проверить вручную
- Открыть сайт на телефоне или в mobile viewport.
- Перейти в "Чат".
- Убедиться, что сначала виден вертикальный список чатов.
- Открыть чат и проверить кнопку "← Чаты".
- Проверить, что поле ввода и кнопка отправки не перекрываются меню.
- Проверить выбор срочности в чате с Будариным.
- Проверить вложения и длинные сообщения.
- Проверить кабинет Бударина на мобильной ширине.
