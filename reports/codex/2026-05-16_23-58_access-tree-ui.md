# Название задачи
Экран "Дерево доступа" в СУП

# Краткое описание задачи
Нужно было оформить раздел управления доступами проекта в стиле макета: группы участников, роли, права, поиск, фильтры, подсказки, действия для администратора и адаптация под мобильную версию.

# Что было изменено
- Вкладка "Структура" переименована в "Дерево доступа".
- Экран дерева доступа теперь доступен участникам выбранного проекта, а не только администратору.
- Управляющие действия показываются только администратору проекта или владельцу системы.
- Участники сгруппированы по `sup_project_members.access_level`: администраторы, менеджеры, участники, наблюдатели.
- Добавлены поиск по участникам и фильтр по ролям.
- Добавлены раскрытие и сворачивание групп.
- Добавлены бейджи ролей с подсказками.
- Добавлены действия "Изменить", "Сохранить", "Отмена", "Удалить" для admin/owner.
- Добавлена frontend-защита от удаления последнего администратора проекта.
- Добавлена статистика по участникам внизу карточки.
- Мобильная версия таблицы превращается в карточки без горизонтальной прокрутки.

# Какие файлы были изменены
- `src/pages/ProjectAccessTree.jsx`
- `src/pages/Projects.jsx`
- `src/index.css`
- `reports/codex/2026-05-16_23-58_access-tree-ui.md`

# Какие ошибки были найдены
- Существующий экран был больше похож на техническую настройку подчинения, а не на понятную таблицу доступа.
- Вкладка дерева была скрыта от не-admin участников, хотя дерево проекта можно показывать участникам без кнопок управления.
- Не было поиска, фильтра и группировки по `access_level`.

# Какие ошибки были исправлены
- Дерево доступа стало полноценным интерфейсом с группами, ролями, правами и статусами.
- Управляющие кнопки теперь условно скрыты для пользователей без прав.
- Добавлена проверка последнего администратора перед удалением.
- Добавлена мобильная адаптация без таблицы, которая ломает ширину.

# Какие проверки выполнены
- Проверка конфликтных маркеров через `git grep`.
- Production-сборка через `npm run build`.
- Проверка доступных scripts в `package.json`.
