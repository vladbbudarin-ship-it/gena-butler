# Projects user plus notice

## 1. Название задачи

Projects user plus notice

## 2. Краткое описание задачи

Добавлен информационный блок для обычных пользователей во вкладке Проекты и скрыты административные действия без прав user_plus/owner.

## 3. Что было изменено

- Добавлена карточка Подключить Пользователь+ с раскрываемой инструкцией
- Скрыта кнопка создания проекта для обычных пользователей
- Проект и задача показываются обычному пользователю в режиме чтения без кнопок сохранения
- AI-контекст для обычного пользователя отображается без кнопки изменения

## 4. Какие файлы были изменены

- src/pages/Projects.jsx
- src/index.css

## 5. Какие ошибки были найдены

- Обычный user видел неактивную кнопку Новый проект вместо понятного объяснения
- Административные формы проекта и задачи показывали disabled-кнопки пользователям без прав

## 6. Какие ошибки были исправлены

- Добавлен понятный сценарий подключения user_plus через Telegram-команду
- Административные кнопки скрыты при отсутствии прав
