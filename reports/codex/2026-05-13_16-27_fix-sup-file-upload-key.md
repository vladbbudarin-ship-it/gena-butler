# Исправление загрузки файлов СУП

## 1. Название задачи

Исправить ошибку `Invalid key` при загрузке файла с кириллическим именем.

## 2. Краткое описание задачи

При загрузке файла `Рамки.png` Supabase Storage отклонял путь `tasks/.../Рамки.png`, потому что ключ объекта содержал недопустимые символы.

## 3. Что было изменено

- Добавлена функция генерации безопасного имени файла для Supabase Storage.
- Для путей Storage теперь используется ASCII-safe имя с `Date.now()` и UUID.
- Исходное имя файла сохраняется в базе в поле `file_name`, поэтому пользователь продолжает видеть русское название файла.
- Исправлена генерация путей и для файлов задач, и для старой функции файлов проекта.

## 4. Какие файлы были изменены

- `src/lib/api.js`
- `reports/codex/2026-05-13_16-27_fix-sup-file-upload-key.md`

## 5. Какие ошибки были найдены

- Supabase Storage получал ключ вида `tasks/<task-id>/<timestamp>-Рамки.png`.
- Кириллица в storage key приводила к ошибке `Invalid key`.

## 6. Какие ошибки были исправлены

- Storage key теперь создаётся в безопасном формате, например `tasks/<task-id>/<timestamp>-<uuid>-file.png`.
- Отображаемое имя файла не меняется.
