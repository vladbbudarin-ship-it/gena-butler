# Регистрация через Telegram

## 1. Название задачи
Добавить регистрацию через Telegram в проект "Дворецкий Гена".

## 2. Краткое описание задачи
Нужно оставить регистрацию по invite-коду и добавить отдельный сценарий регистрации через Telegram Login Widget, при котором Telegram сразу привязывается к профилю.

## 3. Что было изменено
- Добавлена backend-функция регистрации через Telegram.
- Добавлен helper проверки подписи Telegram Login Widget.
- В регистрацию добавлен переключатель "По коду приглашения" / "Через Telegram".
- Добавлен API-вызов `registerWithTelegram`.
- Добавлены небольшие стили для блока подтверждения Telegram.

## 4. Какие файлы были изменены
- `netlify/functions/_utils/telegram-auth.js`
- `netlify/functions/register-with-telegram.js`
- `src/lib/api.js`
- `src/pages/Register.jsx`
- `src/index.css`

## 5. Какие ошибки были найдены
Существующая регистрация поддерживала только invite-код и не имела безопасной backend-проверки Telegram auth data.

## 6. Какие ошибки были исправлены
Frontend больше не является источником доверия для Telegram-данных: backend проверяет HMAC-подпись, срок действия `auth_date`, занятость Telegram ID и email.

## 7. Какие проверки выполнены
- Проверка синтаксиса новых Netlify Functions через `node --check`.
- Проверка конфликтных маркеров.
- `npm run build`.

## 8. Результат npm run build
Успешно.

## 9. Результат тестов, если они есть
E2E-тесты не запускались.

## 10. Что нужно проверить вручную
- На сайте открыть регистрацию и выбрать "Через Telegram".
- Подтвердить Telegram через виджет.
- Заполнить имя, email, пароль и зарегистрироваться.
- Войти по email/password.
- Проверить в профиле статус "Telegram привязан".
- Попробовать повторно зарегистрироваться с тем же Telegram и убедиться, что регистрация запрещена.

## 11. Риски и важные замечания
- Для Telegram Login Widget домен сайта должен быть настроен у бота через BotFather.
- Если username бота изменится, нужно добавить `VITE_TELEGRAM_BOT_USERNAME` в Netlify frontend environment variables.
- SQL не создавался: поля Telegram уже есть в `supabase/telegram-schema.sql`.

## 12. Следующий рекомендуемый шаг
Настроить домен Telegram Login Widget у BotFather и проверить сценарий на опубликованном сайте.
