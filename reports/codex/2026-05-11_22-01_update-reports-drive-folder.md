# Уточнение папки Google Drive для отчётов

## 1. Название задачи

Уточнение папки Google Drive для отчётов

## 2. Краткое описание задачи

Изменено правило размещения копий отчётов: на Google Drive отчёты нужно класть прямо в gena-dvoretskiy/reports без подпапки codex.

## 3. Что было изменено

- Обновлён reports/codex/README.md; уточнён Drive-путь для копий отчётов.

## 4. Какие файлы были изменены

- reports/codex/README.md; reports/codex/<new-report>.md

## 5. Какие ошибки были найдены

- Предыдущее правило указывало Drive-путь gena-dvoretskiy/reports/codex, который больше не нужен.

## 6. Какие ошибки были исправлены

- Drive-путь в README изменён на gena-dvoretskiy/reports/.

## 7. Какие проверки выполнены

- npm.cmd run build.

## 8. Результат npm run build

Успешно: npm.cmd run build завершился без ошибок.

## 9. Результат тестов, если они есть

Тесты не запускались: изменение только в документации отчётов.

## 10. Что нужно проверить вручную

- При ручном переносе отчётов использовать Google Drive папку gena-dvoretskiy/reports/.

## 11. Риски и важные замечания

- Google Drive connector всё ещё не даёт загрузить markdown-файл напрямую в конкретную Drive-папку из Node-скрипта.

## 12. Следующий рекомендуемый шаг

Все следующие отчёты переносить на Google Drive прямо в gena-dvoretskiy/reports/.
